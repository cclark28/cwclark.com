# cwclark.com — Design System

Personal site and AI/interaction testbed for Charlie Clark, senior multidisciplinary designer.

## Folder Structure

```
_design-system/
├── typography/       Type scale, Neue Haas usage, hierarchy rules
├── color-tokens/     CSS custom properties, palette, contrast ratios
├── layout-grid/      Grid specs, breakpoints, spacing scale
├── components/       Nav, buttons, cards, contact form specs
├── icons/            Material Icons usage rules, aria labeling
├── accessibility/    WCAG checklist, focus state specs, ARIA patterns
├── ai-experiments/   Ideas log, experiment briefs, pattern references
└── docs/             This README and general project notes
```
